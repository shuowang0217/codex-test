# STM32F103 有感 FOC 示例（标准外设库）

该示例实现了一个适用于 **STM32F103 + 标准外设库（SPL）** 的三环FOC控制框架：

- **电流内环（Id/Iq PI）**
- **速度中环（PI）**
- **位置外环（PI）**

并集成两类传感器：

- **MA732 绝对角度传感器（SPI）**：用于高速电角度计算。
- **磁栅位置传感器（AB增量编码器 + TIM编码器模式）**：用于机械位置/速度闭环。

## 文件说明

- `inc/foc_control.h`：参数配置、数据结构、接口声明。
- `src/foc_control.c`：FOC核心流程、SPL硬件初始化、三环调节、SVPWM。
- `src/main.c`：启动入口与中断调用示例。

## 使用说明

1. 将代码加入基于 `STM32F10x_StdPeriph_Lib` 的工程。
2. 根据硬件修改 GPIO、ADC 通道、死区时间、极对数、编码器线数等参数。
3. 在 `TIM1_UP_IRQHandler` 中以固定频率调用 `FOC_ControlLoop_ISR()`。
4. 通过上位机或串口修改 `g_foc_state.position_ref`、PI参数进行整定。

> 注意：示例偏重控制结构与可读性，实际产品请补充过流/过压/堵转/传感器异常等保护。
